WebLauncher

  • Это легковесное ПО для загрузки файлов клиента, является встроенной в CMS с открытым исходным кодом - SphereWeb.

  • Единажды скаченный лаунчер, подойдет для всех аккредитованных серверов на CMS SphereWeb.

  • Лаунчер для обновления игры, является мощным инструментом благодаря использованию передовых технологий.

  • Программа основана на принципах многопоточности, конкурентности и других возможностях. Это позволяет лаунчеру скачивать файлы одновременно и много, обеспечивая быструю обработку и эффективное использование ресурсов компьютера.

  • Одним из главных преимуществ лаунчера является использование передовой технологии, которая позволяет работать с ней в браузере. Это делает использование лаунчера удобным и доступным для всех пользователей.

  • Еще одним важным преимуществом вашего лаунчера является то, что он не потребляет много памяти и работает быстрее, чем его конкуренты. Это обеспечивает более быструю и эффективную работу лаунчера и повышает удобство его использования.

  • Кроме того, ваш лаунчер позволяет пользователям использовать только один лаунчер для всех серверов и всех хроник, что значительно упрощает и ускоряет процесс обновления игры.

  • Пользователю не придется загружать для каждого сервера лаунчер, достаточно зайти на сайт сервера.

  • В целом, лаунчер - это мощный и удобный инструмент для обновления игры, который предлагает передовые технологии и обеспечивает быструю и эффективную работу.

    • Как всё работает?

    После того как Вы скачаете лаунчер и запустите его, лаунчер сможет принимать и обрабатывать команды в реал-тайме.

    Лаунчер создаст протокол для удобного запуска из браузера.

    Для этого на сайте сервера есть интерфейс для обновления, он Вам доступен и все его возможности.

    10.04.2023

    Информация о первой альфа-версии.

    • Массовые загрузки (5 одновременных загрузок)
    • Админы могут создавать патчи прямо через SphereWeb CMS при включенном лаунчере
    • Выбор места загрузки файлов клиента
    • Вывод ошибки загрузки и прерывание загрузки, если http статус код не 200
    • Дополнительные доработки для ускоренния архивирования и распаковка файлов.
    • К файлам менее 50кб не применяется функция архивации.
    • Появился значок лаунчера в трее.
